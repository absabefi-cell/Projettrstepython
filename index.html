Projet Python : Calcul d'une Sphère

Ce projet calcule le volume et la surface d'une sphère en utilisant Python.

Code principal :

import math

r = float(input("Rayon de la sphère : "))
volume = 4/3 * math.pi * r**3
surface = 4 * math.pi * r**2

print("Volume :", volume)
print("Surface :", surface)

Exécution

Pour lancer le projet :

python3 sphere1.py
